    Link to User Stories

    #25 , User flow 7 in Figma

    Objective

    • Implement automatic QR Code Generation for users Upon Sign Up.
    • Link it to the Firebase User database (Create User Database #14)

    Acceptance Criteria

    • Each user should have a unique QR code generated within the app.
    • Users should have the ability to access their QR code easily within the app for quick event check-ins, from the Bottom Navigation Bar option "QR Code".
    • QR codes should be non-transferable and linked directly to the user's account to prevent unauthorized access.
    • Measures should be implemented to ensure the security and authenticity of the QR codes.
